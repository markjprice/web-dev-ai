> **Prompt**: Please show me an example of a bad one-shot prompt for TallyApp and how to improve it.

Because "Tally" can refer to two very different (but popular) tools in 2026—**Tally.so** (the document-style form builder) and **TallyPrime** (the enterprise accounting suite)—the biggest mistake in a "one-shot" prompt is **ambiguity**.

A one-shot prompt is a single instruction where the AI has one chance to get it right. If you aren't specific about the tool, the context, and the structure, the AI will guess, often leading to a generic or unusable result.

---

## ❌ The Bad One-Shot Prompt
> **"Make me a survey for Tally."**

### Why this fails:
1.  **Tool Ambiguity:** Does "Tally" mean the form builder or the accounting software?
2.  **No Context:** What is the survey for? Is it for employees, customers, or a wedding?
3.  **Missing Structure:** Does it need conditional logic? A specific number of pages? File uploads?
4.  **No Brand/Tone:** Should it be professional, casual, or a "Buzzfeed-style" quiz?

---

## ✅ The Improved One-Shot Prompt
To fix this, use the **C.R.E.A.T.E.** framework (Context, Role, Entity, Action, Tone, Evaluation).

> **"Act as a Product Researcher (Role). Create a customer feedback form for Tally.so (Entity/Context) for a new SaaS project. I need a multi-page layout (Structure) including: a Net Promoter Score (NPS) scale, a conditional logic block that asks 'What can we improve?' only if the score is below 7, and a file upload field for screenshots (Action). Keep the tone professional but encouraging (Tone). Format the output as a list of Tally 'blocks' (e.g., /question, /rating) that I can copy-paste (Evaluation)."**

### Why this works:
* **Zero Ambiguity:** Specifies **Tally.so** and the exact "blocks" format.
* **Logic-Ready:** It explicitly mentions **conditional logic**, which is a core feature of Tally.so.
* **Actionable:** It tells the AI exactly what fields to include, preventing "hallucinated" features.

## Comparison: Result Quality

| Feature | Result from "Bad" Prompt | Result from "Improved" Prompt |
| :--- | :--- | :--- |
| **Logic** | Static list of 5 generic questions. | Dynamic flow with "Skip Logic" for low scores. |
| **Integration** | None. | Structured for easy Notion/Airtable sync. |
| **Accuracy** | Might give you accounting ledger tips. | Gives you exact `/block` commands for the form. |

### Pro-Tip for TallyPrime (Accounting)
If you meant the accounting software, your one-shot prompt must include **Tax Jurisdictions** and **Voucher Types**:
> *"Generate a step-by-step guide for TallyPrime 2026 to set up a GST-compliant sales voucher for a service-based business in Maharashtra, including HSN codes and automated CGST/SGST calculations."*
