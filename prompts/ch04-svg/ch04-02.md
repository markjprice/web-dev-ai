To understand SVG styling, it helps to see how small changes in attribute values radically alter the "skeleton" of a shape. In SVG, **Line Caps** define how the ends of a line look, while **Line Joins** define how the corners where two lines meet are rendered.

Below are the live code examples you can drop into any HTML file.

---

## 1. Stroke Line Caps

The `stroke-linecap` attribute determines the shape used at the end of open subpaths.

```html
<svg width="400" height="200" viewBox="0 0 400 200" style="border: 1px solid #ccc;">
  <line x1="50" y1="50" x2="350" y2="50" stroke="black" stroke-width="20" stroke-linecap="butt" />
  <text x="50" y="30" font-family="Arial" font-size="14">butt (Default)</text>

  <line x1="50" y1="100" x2="350" y2="100" stroke="blue" stroke-width="20" stroke-linecap="round" />
  <text x="50" y="80" font-family="Arial" font-size="14">round</text>

  <line x1="50" y1="150" x2="350" y2="150" stroke="red" stroke-width="20" stroke-linecap="square" />
  <text x="50" y="130" font-family="Arial" font-size="14">square</text>
  
  <line x1="50" y1="20" x2="50" y2="180" stroke="#FFD700" stroke-width="1" />
  <line x1="350" y1="20" x2="350" y2="180" stroke="#FFD700" stroke-width="1" />
</svg>

```

<svg width="400" height="200" viewBox="0 0 400 200" style="border: 1px solid #ccc;">
  <line x1="50" y1="50" x2="350" y2="50" stroke="black" stroke-width="20" stroke-linecap="butt" />
  <text x="50" y="30" font-family="Arial" font-size="14">butt (Default)</text>

  <line x1="50" y1="100" x2="350" y2="100" stroke="blue" stroke-width="20" stroke-linecap="round" />
  <text x="50" y="80" font-family="Arial" font-size="14">round</text>

  <line x1="50" y1="150" x2="350" y2="150" stroke="red" stroke-width="20" stroke-linecap="square" />
  <text x="50" y="130" font-family="Arial" font-size="14">square</text>
  
  <line x1="50" y1="20" x2="50" y2="180" stroke="#FFD700" stroke-width="1" />
  <line x1="350" y1="20" x2="350" y2="180" stroke="#FFD700" stroke-width="1" />
</svg>


> **Note:** Notice how `round` and `square` actually extend the line beyond its X coordinates, whereas `butt` stops exactly on the line.

## 2. Stroke Line Joins

The `stroke-linejoin` attribute defines how the corners of shapes (like polylines or paths) are handled.

```html
<svg width="400" height="250" viewBox="0 0 400 250" style="border: 1px solid #ccc;">
  <polyline points="50,80 100,20 150,80" fill="none" stroke="black" stroke-width="15" stroke-linejoin="miter" />
  <text x="60" y="110" font-family="Arial" font-size="14">miter</text>

  <polyline points="175,80 225,20 275,80" fill="none" stroke="blue" stroke-width="15" stroke-linejoin="round" />
  <text x="190" y="110" font-family="Arial" font-size="14">round</text>

  <polyline points="300,80 350,20 400,80" fill="none" stroke="red" stroke-width="15" stroke-linejoin="bevel" />
  <text x="315" y="110" font-family="Arial" font-size="14">bevel</text>
</svg>

```

### Summary of Differences

| Property | Value | Visual Result |
| --- | --- | --- |
| **Line Cap** | `butt` | Flat edge, ends exactly at the point. |
|  | `round` | Rounded edge, extends past the point. |
|  | `square` | Squared edge, extends past the point. |
| **Line Join** | `miter` | Sharp, pointed corner (can be limited by `miterlimit`). |
|  | `round` | Smooth, curved corner. |
|  | `bevel` | Flat, "clipped" corner. |
