Introduction to Web Development Workshop

This one-day workshop introduces the building blocks of the web. You’ll learn how web pages are structured with HTML and how browsers interpret that structure to display content like headings, lists, tables, images, and forms.

Agenda

We’ll move step-by-step through the basics and finish with a mini project.

  1. How the web works: pages, browsers, and servers
  2. HTML essentials: structure, headings, paragraphs, and links
  3. Working with images, lists, and tables
  4. Forms and input elements: collecting user information
  5. Mini build: create a simple multi-page site

What you’ll learn

  • How to structure a page using headings and paragraphs
  • How to connect pages using links
  • How to include images with meaningful alt text
  • How to present data clearly using tables
  • How to design a basic form using input elements
